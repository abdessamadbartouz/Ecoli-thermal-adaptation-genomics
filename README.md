# Ecoli-thermal-adaptation-genomics
Reproducible bioinformatics workflow for whole-genome analysis of experimentally evolved Escherichia coli adapting to prolonged heat stress.
# Computational Characterization of Mutation Dynamics and Functional Adaptation in Experimentally Evolved *Escherichia coli* Under Heat Stress

**Author:** Abdessamad Bartouz

## Overview

This repository contains the bioinformatics workflow and analysis scripts developed during my Master's thesis entitled:

**Computational Characterization of Mutation Dynamics and Functional Adaptation in Experimentally Evolved *Escherichia coli* Under Heat Stress**

The project investigates the genomic basis of thermal adaptation in experimentally evolved *Escherichia coli* DH10B using whole-genome sequencing (WGS). A complete and reproducible bioinformatics workflow was developed to identify genomic variants, characterize mutation dynamics, perform functional enrichment analyses, and visualize adaptive evolutionary patterns.

---

## Project Background

Experimental evolution is a powerful approach for studying microbial adaptation under controlled environmental conditions. In this study, *Escherichia coli* DH10B populations were evolved under progressively increasing temperature conditions, allowing the identification of genetic changes associated with long-term heat adaptation.

The sequencing dataset consists of isolates collected throughout the evolution experiment:

| Isolate | Evolution Time | Growth Temperature |
|----------|---------------:|-------------------:|
| DH10B | 0 days | 43.5°C |
| HE50 | 50 days | 45.5°C |
| HE100 | 100 days | 45.7°C |
| HE150 | 150 days | 46.9°C |

The maximum temperature tolerated by the evolved population was **46.9°C**.

## Bioinformatics Workflow

The analysis pipeline consists of the following major steps:

```
Raw FASTQ files
        │
        ▼
Quality Control (FastQC, MultiQC)
        │
        ▼
Read Mapping (BWA-MEM)
        │
        ▼
Variant Calling
   ├── BCFtools
   └── iVar
        │
        ▼
Variant Filtering
        │
        ▼
Chromosome Renaming
        │
        ▼
Variant Annotation (SnpEff)
        │
        ▼
Mutation Matrix Construction
        │
        ▼
Functional Enrichment Analysis
        │
        ▼
Clustering & Visualization
```

A graphical version of the workflow is available in the **Docs/** directory.

## Analysis

The workflow includes:

- Quality assessment of sequencing reads
- Read alignment against the *E. coli* DH10B reference genome
- Variant calling using two independent callers
- Variant filtering
- Functional annotation
- Binary SNP matrix construction
- Mutation dynamics analysis
- Hierarchical clustering
- Principal Component Analysis (PCA)
- Venn diagram analysis
- Functional enrichment analysis (DAVID and ShinyGO)
- Publication-quality visualizations
